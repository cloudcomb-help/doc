# 删除自定义镜像

<span>Note:</span>
自定义镜像删除后不可恢复；
不影响使用该镜像创建的实例。

## 操作步骤

1. 登录 [控制台](https://c.163.com/dashboard#/m/win/)，定位到「**云主机**」标签；
2. 点击右侧「**自定义云主机镜像**」按钮：
![](../../image/使用指南-镜像-自定义镜像.png)
3. 在详情页定位到目标镜像，点击右侧操作列的「**删除**」按钮；
4. 确认删除：
![](../../image/使用指南-镜像-自定义镜像删除.png)
