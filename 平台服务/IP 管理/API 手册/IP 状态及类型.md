# IP 管理

## IP 状态

|    状态   |  描述  |        备注        |
|-----------|--------|--------------------|
| available | 可用   | 在该状态下可被释放 |
| disabled  | 禁用   | 网络服务不可用     |
| binding   | 绑定中 | -                  |
| binded    | 已绑定 | -                  |

### IP 类型

IP 类型表示该 IP 只能由该类型的实例绑定。

| 类型 |     描述     |         备注         |
|------|--------------|----------------------|
| nce  | 通用服务类型 | 可以绑定服务、云主机 |
| nlb  | 负载均衡类型 | 可以定负载均衡       |
