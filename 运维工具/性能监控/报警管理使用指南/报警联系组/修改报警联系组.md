# 修改报警联系组

## 操作步骤

1. 登录 [控制台](https://c.163.com/dashboard#/m/apm/alarm/)，点击「**监控**」导航，点击「**报警管理**」标签；
2. 在右侧点击「**报警组管理**」按钮：
![](../../image/使用指南-报警组管理.png)
3. 点击「**报警联系组**」标签；
4. 在右侧操作列，点击「**修改**」按钮：
![](../../image/使用指南-报警组管理-修改报警联系组.png)
5. 支持修改备注，增删报警联系人：
![](../../image/使用指南-报警组管理-修改报警联系组-详情.png)
